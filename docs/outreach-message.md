# Outreach Message

Use or adapt this short message when inviting contributors to submit projects or improve the catalogue.

## Short Version

We are building **Open Source Madagascar**, a curated repository of open source projects, datasets, tools, and starter kits useful for the Malagasy context.

If you know a project related to payments, education, delivery, public services, language AI, offline-first software, or other Madagascar-focused needs, we would love to include it.

You can contribute by opening an issue or pull request with a project submission:

- repository link
- short description
- category
- language
- license
- status
- clear Madagascar relevance

The goal is simple: make local digital solutions easier to discover, reuse, and improve.

## Community Post Version

Open Source Madagascar is a public catalogue of open source projects relevant to Madagascar. We are collecting useful repositories, datasets, developer tools, and local digital resources so builders, students, NGOs, startups, and institutions can find and reuse them more easily.

If you know a good project, send it through a GitHub issue or pull request. Even a single strong submission helps make the ecosystem more visible.

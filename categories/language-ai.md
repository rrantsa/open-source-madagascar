# Language and AI

This category will list open source projects for Malagasy NLP, OCR, translation, speech-to-text, language datasets, and related AI tooling.

## Included Projects

### Gasy Corpus Crawler

- Repository: https://github.com/SetraC4Ci/Gasy-Corpus-Crawler
- Description: Python crawler toolkit for scraping Malagasy-language websites and building text corpora for dataset creation.
- Category: language-ai
- Language: Python
- Framework: BeautifulSoup and Selenium
- License: MIT
- Status: experimental
- Maintainer: SetraC4Ci
- Target users: Researchers, NLP builders, dataset curators, developers
- Madagascar relevance: The repository is specifically designed to scrape Malagasy-language websites and build Malagasy text corpora, making it directly useful for local NLP, language modeling, and corpus creation work.
- Notes: The README presents the project as a set of scripts for corpus scraping with both BeautifulSoup and Selenium variants. The small commit history and script-oriented design suggest an experimental maturity level.

### MMCLI

- Repository: https://github.com/HarimbolaSantatra/mm-cli
- Description: Command-line Malagasy encyclopedia and dictionary client that queries Malagasyword.org and returns searchable word data.
- Category: language-ai
- Language: Go
- Framework: Beautiful Soup parsing helper
- License: GPL-3.0
- Status: experimental
- Maintainer: HarimbolaSantatra
- Target users: Developers, students, language learners, CLI users
- Madagascar relevance: The tool is built specifically around Malagasy dictionary and encyclopedia lookup, which makes it directly relevant to Malagasy language access and language-tooling workflows.
- Notes: The README states that the latest release is the only stable version and that the rest remains a work in progress with unresolved issues, so `experimental` is the safest catalogue status.

### Rakibolana Malagasy

- Repository: https://github.com/Rohan29-AN/rakibolana_malagasy
- Description: Dataset repository containing Malagasy words and their meanings in CSV, JSON, and SQLite formats.
- Category: language-ai
- Language: JSON
- Framework: null
- License: unknown
- Status: unknown
- Maintainer: Rohan29-AN
- Target users: Developers, students, researchers, language learners

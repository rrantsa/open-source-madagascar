# Civic Tech

This category will list open source projects that help communities report issues, coordinate locally, access public information, or improve civic participation in Madagascar.

## Included Projects

### VeriSphere

- Repository: https://github.com/Hernat/VeriSphere
- Description: Android floating bubble app that verifies social-media posts through Gemini in one long-press and expands into a source-backed detail panel.
- Category: civic-tech
- Language: Kotlin
- Framework: Android, Jetpack Compose, and Gemini API
- License: MIT
- Status: experimental
- Maintainer: Hernat
- Target users: Android users, journalists, students, civic-tech contributors
- Madagascar relevance: An Android-first fact-checking tool can help Malagasy users, journalists, and community organizations assess viral social-media claims more quickly in a mobile-first information environment.
- Notes: The repository has a clear MIT license and visible activity on 2026-05-20. The README explicitly labels the app as a pre-V1 alpha at version `0.1.0`, with no signed APK published yet, so `experimental` is the safest catalogue status.

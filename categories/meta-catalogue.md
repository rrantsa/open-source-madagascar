# Meta Catalogue

This category lists catalogue, discovery, and curation resources that help people find Madagascar-related open source projects more efficiently.

## Included Projects

### Awesome Madagascar

- Repository: https://github.com/luckasRanarison/awesome-madagascar
- Description: Curated Markdown list of projects, tools, websites, and resources related to Madagascar.
- Category: meta-catalogue
- Additional categories: developer-tools
- Language: Markdown
- Framework: Markdown awesome list
- License: CC0-1.0
- Status: active
- Maintainer: luckasRanarison
- Target users: Developers, students, researchers, ecosystem mappers
- Madagascar relevance: The repository serves as a discovery layer for Madagascar-related projects and helps maintain ongoing ecosystem visibility across multiple technical domains.
- Notes: The repository is close to this catalogue's mission but complementary rather than duplicative, making it a strong source for watchlist-style curation.

- Category: meta-catalogue
- Additional categories: community
- Language: TypeScript
- Framework: React and Vite
- License: unknown
- Status: inactive
- Maintainer: tsirysndr
- Target users: Developers, students, recruiters, ecosystem mappers
- Madagascar relevance: The project supports ecosystem discovery by making Malagasy GitHub contributors easier to find, which helps collaboration, visibility, and community mapping.

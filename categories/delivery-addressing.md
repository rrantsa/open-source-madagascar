# Delivery and Addressing

This category will list open source projects related to local addressing formats, GPS support, delivery management, route coordination, and location systems adapted to Madagascar.

## Included Projects

### Mada CLI

- Repository: https://github.com/tsirysndr/mada
- Description: CLI and companion web tooling for searching Malagasy regions, districts, communes, and fokontany.
- Category: delivery-addressing
- Additional categories: developer-tools
- Language: Go and TypeScript
- Framework: CLI and web tooling
- License: BSD-3-Clause
- Status: active
- Maintainer: tsirysndr
- Target users: Developers, delivery services, mapping builders, public-service tool builders
- Madagascar relevance: The project is built around Madagascar's administrative and locality structure, including regions, districts, communes, and fokontany, which is directly useful for local search, address handling, and territorial data workflows.
- Notes: The repository shows visible maintenance through 2026 and combines reusable territorial lookup with a developer-friendly CLI, making it a strong early catalogue entry.

### Zip Code Madagascar

- Repository: https://github.com/rasolofonirina/zip-code-madagascar
- Description: Small JSON dataset of Madagascar postal codes for validation and address-oriented forms.
- Category: delivery-addressing
- Additional categories: datasets
- Language: JSON
- License: MIT
- Status: inactive
- Maintainer: rasolofonirina
- Target users: Developers, form builders, delivery services, civic platforms
- Madagascar relevance: The dataset is directly useful for Madagascar-specific address validation, lookup, and form completion workflows.
- Notes: The repository is simple and immediately reusable. Its last visible code push was on 2022-05-24, so `inactive` is safer than assuming active maintenance.

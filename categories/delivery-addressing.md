# Delivery and Addressing

This category will list open source projects related to local addressing formats, GPS support, delivery management, route coordination, and location systems adapted to Madagascar.

## Included Projects

### Mada CLI

- Repository: https://github.com/tsirysndr/mada
- Description: CLI for searching Malagasy regions, districts, communes, and fokontany, with support for local geographic lookup and a web UI.
- Category: delivery-addressing
- Language: Go
- Framework: Spatialite-backed CLI and web tooling
- License: BSD-3-Clause
- Status: stable
- Maintainer: tsirysndr
- Target users: Developers, delivery services, mapping builders, public-service tool builders
- Madagascar relevance: The project is built around Madagascar's administrative and locality structure, including regions, districts, communes, and fokontany, which is directly useful for local search, address handling, and territorial data workflows.
- Notes: The repository has multiple releases, installation instructions, and a mature usage section, which supports a `stable` catalogue status.

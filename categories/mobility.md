# Mobility

This category lists open source projects related to transport, routing, traffic information, ride coordination, and other movement-oriented services relevant to Madagascar.

## Included Projects

### Signaleo

- Repository: https://github.com/julkwel/signaleo
- Description: Mobile application for sharing street, traffic, and carpooling information in Madagascar.
- Category: mobility
- Additional categories: public-services, civic-tech
- Language: TypeScript
- Framework: Ionic and Angular mobile app
- License: MIT
- Status: inactive
- Maintainer: julkwel
- Target users: Drivers, commuters, civic-tech builders, local mobility startups
- Madagascar relevance: The project is strongly localized around everyday mobility and congestion-sharing needs in Madagascar, which makes it a good fit for both mobility and civic-tech discovery.
- Notes: The repository remains valuable as a localized mobility reference, but its last visible code push was on 2023-11-12, so `inactive` is the safest catalogue status.

### BetaX Community

- Repository: https://github.com/puchka/BetaX-Community
- Description: Scripts and generated route assets for reconstructing bus paths in the urban area of Antananarivo from waypoint data.
- Category: mobility
- Additional categories: datasets
- Language: Python and HTML
- Framework: Google Maps waypoint and directions workflow
- License: unknown
- Status: inactive
- Maintainer: puchka
- Target users: Commuters, mobility researchers, mapping builders, civic-tech contributors
- Madagascar relevance: The repository is centered on Antananarivo bus routes and provides reusable route-generation artifacts for a concrete local mobility problem.
- Notes: The repository includes waypoint inputs, generated route outputs, and small helper scripts for mapping bus trajectories. No reusable license file was present during review, so the license is catalogued as `unknown`. The last visible code push was on 2017-05-01, so `inactive` is the safest status.
